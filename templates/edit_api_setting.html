Security Information
API Key Security

Your API keys are stored securely in the database with encryption. However, please follow these best practices:

  • Never share your API keys publicly
  • Use unique keys for different applications
  • Rotate keys periodically for security
  • Do not commit API keys to version control
This application stores your API keys securely using Django's built-in security features. They are never exposed in client-side code or logs.
